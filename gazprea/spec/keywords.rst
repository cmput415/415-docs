Keywords
========

*Gazprea* has a number of built in keywords that are reserved and should
not be used by a programmer.

-  and

-  as

-  boolean

-  break

-  by

-  call

-  character

-  columns

-  const

-  continue

-  else

-  false

-  filter

-  function

-  identity

-  if

-  in

-  integer

-  interval

-  length

-  loop

-  matrix

-  not

-  null

-  or

-  procedure

-  real

-  return

-  returns

-  reverse

-  rows

-  std_input

-  std_output

-  stream_state

-  string

-  true

-  tuple

-  typedef

-  var

-  vector

-  while

-  xor
