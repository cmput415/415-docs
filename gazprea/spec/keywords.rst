Keywords
========

*Gazprea* has a number of built in keywords that are reserved and should
not be used by a programmer.

-  and

-  as

-  boolean

-  break

-  by

-  call

-  character

-  columns

-  const

-  continue

-  else

-  false

-  format

-  function

-  if

-  in

-  integer

-  length

-  loop

-  not

-  or

-  procedure

-  real

-  return

-  returns

-  reverse

-  rows

-  std_input

-  std_output

-  stream_state

-  string

-  true

-  tuple

-  typealias

-  var

-  vector

-  while

-  xor
